How to classify content

Tabs organize items by the decision you have made. You can change a decision from any tab.

Add: this item should be included in the curated feed.

Ignore: this specific item is not useful. Pick a reason so discarded content can be analyzed later.

Auto ignored: items discarded by rules or AI. Use Add when the algorithm made a mistake; export the JSONL so the next run can keep it.

Block similar: this item exposes a pattern to automate later, such as a domain, URL type, job post, spam, or datasets that should be filtered in future runs.

Sync from file: clears local browser decisions and reloads the baseline saved in data/curation_decisions.jsonl.
